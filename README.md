# Everything Checker v2.0 | RustMe Edition 🛡️

**Everything Checker** — это профессиональный инструмент для глубокого анализа системы на наличие читов, лоадеров и спуферов. Специально оптимизирован для проверки игроков на проекте **RustMe**.

Программа использует уникальные байтовые подписи (Signatures) и эвристический анализ, что позволяет находить даже переименованные и скрытые угрозы.

---

## 🚀 Основные возможности

✅ **Детекция Akcel & Nemezida:** Поиск лоадеров и спуферов по уникальным внутренним сигнатурам.  
✅ **Deep Scan (10MB):** Глубокий анализ содержимого EXE/DLL/JAR файлов (чтение первых 10 МБ для гарантированного детекта).  
✅ **Детекция Спуферов:** Выявление попыток обхода бана по HWID (типы `Spoof(A)` и `Spoof(N)`).  
✅ **Анализ конфигов:** Поиск обфусцированных `.cfg` файлов читов в `LocalAppData` (детект по специфическим символам).  
✅ **Проверка процессов:** Мониторинг активных окон и процессов в реальном времени.  
✅ **Виртуальные диски:** Обнаружение скрытых лоадеров, работающих через виртуальные или сетевые диски.  
✅ **Last Report:** Возможность мгновенно открыть последний отчет без повторного сканирования.  
✅ **Приоритетная сортировка:** Лоадеры и Спуферы всегда отображаются в самом верху списка.

---

## 🧠 Технологии проверки

Программа выполняет комплексный анализ по нескольким направлениям:

### 1️⃣ Сигнатурный анализ
Поиск уникальных последовательностей байтов (signatures), присущих конкретным читам. Это позволяет идентифицировать файл, даже если его название — `calculator.exe`.

### 2️⃣ Эвристика конфигов
Автоматическое распознавание "китайской" обфускации в настройках читов Akcel и Nemezida.

### 3️⃣ Сканирование критических папок
Целевая проверка мест, где чаще всего прячут читы:
- `Downloads` (включая Telegram Desktop)
- `AppData` (.minecraft, RustMe, Temp)
- `Desktop`
- `Windows Recent` (история последних запусков)

### 4️⃣ Анализ артефактов RustMe
Проверка целостности игровых папок RustMe на наличие сторонних инъекций и подозрительных DLL.
